HRMS Pro – Employee Management System
A modern, enterprise-grade Human Resource Management System (HRMS) built with Next.js, TypeScript, Tailwind CSS, ShadCN UI, and Framer Motion.
HRMS Pro is a fully responsive SaaS-style employee management platform featuring employee administration, attendance tracking, payroll analytics, recruitment management, performance monitoring, training modules, and business intelligence dashboards.
