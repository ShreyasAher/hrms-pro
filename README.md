# HRMS Pro – Employee Management System

A modern, enterprise-grade Human Resource Management System (HRMS) built with Next.js, TypeScript, Tailwind CSS, ShadCN UI, and Framer Motion.

---

## Features

### Dashboard

* Executive overview dashboard
* Employee statistics and KPIs
* Department distribution analytics
* Growth and attendance charts
* Company announcements and activity feeds

### Employee Management

* Employee directory
* Employee profile pages
* Department assignment
* Role and designation management
* Search and filtering capabilities

### Attendance Management

* Attendance dashboard
* Clock-in / Clock-out interface
* Attendance analytics
* Weekly and monthly attendance insights
* Attendance history tracking

### Leave Management

* Leave application workflow
* Leave balance tracking
* Leave history management
* Holiday calendar

### Payroll Management

* Payroll overview dashboard
* Salary analytics
* Payslip generation interface
* Payroll reporting

### Payment Module

* Razorpay-inspired payment UI
* Stripe-inspired payment UI
* Subscription plans
* Invoice management
* Payment history tracking

### Department Management

* Department overview
* Budget allocation visualization
* Department performance analytics
* Team management insights

### Performance Management

* KPI dashboards
* Goal tracking
* Employee performance analytics
* Rating trends and reports

### Recruitment Management

* Job listings
* Candidate pipeline tracking
* Interview scheduling interface
* Recruitment analytics

### Training & Learning

* Course catalog
* Employee learning progress
* Skills tracking
* Certification management

### Reports & Analytics

* Interactive charts
* Workforce analytics
* Attendance reporting
* Payroll reporting
* Department insights

### Settings & Profile

* User profile management
* Theme customization
* Notification preferences
* Security settings

## Design System

This application follows a premium **Neumorphic (Soft UI)** design language featuring:

* Soft shadows and depth effects
* Rounded interfaces
* Light and Dark themes
* Smooth micro-interactions
* Animated transitions
* SaaS-style enterprise layout

## Installation

```bash
git clone https://github.com/your-username/hrms-pro.git

cd hrms-pro

npm install

npm run dev
```

Application runs at:

http://localhost:3000
